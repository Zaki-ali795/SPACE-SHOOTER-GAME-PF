# Space Shooter 2D Game
## Description about Project
This Space Shooter Game is a normal copy of original Space Shooter in which  our SpaceShip and Enemies are represented by rectangles.<br>
Its a beginner level project which I had completed with my college partner so it may have some incompetencies but it works fine.<br>
Its uses SFML and C++ with concepts of OOP.
### Game Description
This Game has an intro screen with complete description of the controls and game rules .
We go to next level after every 100 score . 
In Each Level speed of Enemies ( represented as rectangles ) coming down increases which makes difficulty of player to kill them .
We have 3 health and health decreases if enemies collide with our spaceship or with our base . 
If we lost all our health , A Gameover screen is displayed with your current score. You can restart the game from level 1.
If we manage to kill all the enemies till level 5 , a Winning screen is displayed with highest Score .
